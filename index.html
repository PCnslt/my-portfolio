About Me

With over 14 years in quality engineering and test automation, I specialize in building scalable frameworks that bridge modern development with seamless testing. From architecting CI/CD pipelines to automating both web and mobile UI, I’ve led QA efforts that span cloud-native ecosystems, machine learning platforms, and enterprise-level applications.

My toolbox includes TypeScript, Java, and Python—delivered through frameworks like Cypress, Selenium, Appium, and BDD-driven architectures. I've written data-driven test scripts, managed test execution in CI/CD pipelines, and automated validations for REST APIs, UIs, and backend workflows.

I’ve engineered solutions using AWS services like Lambda, S3, DynamoDB, and CloudFormation, and automated test data pipelines with Python’s boto3. I track KPIs using dashboards and generate actionable QA reports to close feedback loops across fast-paced Agile teams.

Whether it’s defining QA strategy, mentoring automation engineers, or implementing DevOps-first practices, I deliver quality at scale—through code, process, and collaboration.
